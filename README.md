# AjLisp in Scala

Lisp interpreter written in Scala, WIP.

## Contribution

Feel free to [file issues](https://github.com/ajlopez/AjLispScala) and submit
[pull requests](https://github.com/ajlopez/AjLispScala/pulls) — contributions are
welcome.

If you submit a pull request, please be sure to add or update corresponding
test cases, and ensure that tests continue to pass.
